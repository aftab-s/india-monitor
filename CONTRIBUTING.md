# Contributing to Observe India

First off, thank you for considering contributing to Observe India! It's people like you that make the open-source community such an amazing place to learn, inspire, and create.

## 🚀 How Can I Contribute?

### Reporting Bugs
- Use the **Issues** tab to report bugs.
- Include a clear title and description.
- Provide as much relevant information as possible (OS, Browser, steps to reproduce).

### Suggesting Enhancements
- Enhancement suggestions are tracked as GitHub issues.
- Explain why this enhancement would be useful to most Observe India users.

### Pull Requests
1. **Fork** the repo and create your branch from `main`.
2. If you've added code that should be tested, **add tests**.
3. If you've changed APIs, **update the documentation**.
4. Ensure the test suite passes.
5. Make sure your code follows the existing style (Technical Wireframe aesthetic).

## 🎨 Design Guidelines
Observe India follows a strict technical design language:
- **Responsive Panels**: Ensure all new panels support dynamic resizing and clean grid wrapping.
- **Monochrome Only**: Use gray scales unless indicating a specific status (Red/Green/Amber).
- **Square Edges**: Avoid rounded corners to maintain a professional, systematic look.
- **Mono Fonts**: Use monospace fonts for all data points and technical labels.
