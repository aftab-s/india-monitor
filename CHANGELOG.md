# Changelog

All notable changes to the **Observe India** project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Dynamic National News Streaming integration for YouTube live feeds.
- Automated sync scripts for YouTube, Fuel Prices, and EV Stations.
- Caching pipeline using Vercel Blob for EV and Fuel data.

## [1.1.0] - 2026-05-11

### Added
- State-level fuel price integration.
- Draggable and resizable panels for custom dashboard layouts (Nothing OS aesthetic).
- Geospatial hover interactions displaying capital cities on the map.

## [1.0.0] - 2026-05-04

### Added
- Core Intelligence Pipeline with Groq AI integration.
- Real-time weather and AQI monitoring via Open-Meteo.
- Financial indicators (NIFTY/SENSEX, Currency).
- Wireframe map engine using MapLibre GL.
- Centralized dataset in `news.json`.
